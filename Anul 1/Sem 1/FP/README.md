# FP - Fundamentele Programării

Laboratoare, probleme și simulări de examen realizate în Python în semestrul 1, anul I, Informatică (UBB Cluj-Napoca). Majoritatea proiectelor urmează arhitectura pe straturi: `domain` / `repo` / `controller` (sau `service`) / `ui`.

## Structură
- `lab3.py`, `lab 4_6`, `Lab7_9` - laboratoare săptămânale
- `Prob1`, `Prob2`, `Prob3` - probleme rezolvate cu arhitectură pe straturi
- `practic`, `prob_exersare` - exerciții practice
- `Backtracking` - exerciții de recursivitate/backtracking
- `Simulare_FP` - simulare examen
- `seminar.py`, `seminar_sapt4.py`, `sem2.py` - exerciții de seminar
- `Documentatie lab*.docx` - documentații de laborator
