# ASC - Arhitectura Sistemelor de Calcul

Laboratoare, seminarii, teme și exerciții de assembly (NASM) și interoperabilitate C-ASM, realizate în semestrul 1, anul I, Informatică (UBB Cluj-Napoca).

## Structură
- `Lab3` ... `Lab10` - laboratoare săptămânale
- `SeminarSapt4`, `Seminar sapt 14` - exerciții de seminar
- `Temelab` - temă de laborator
- `Problema 1/2/3` - probleme rezolvate
- `Exercitii test 2`, `Teste` - pregătire/exerciții de test
- `Practic`, `Practicex`, `Exercitiu Practic` - exerciții practice C + ASM
- `ASM_tools` - unelte folosite (NASM, OllyDbg, Notepad++)
- `main.c`, `CMakeLists.txt` - exercițiu C independent

## Tehnologii
Assembly x86 (NASM), C
